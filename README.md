# Cinema Pergamino - Sistema de Gestión de Cine

Sistema completo para gestión de cine con panel administrativo, venta de entradas y kiosco.

## Estructura del Proyecto

- `backend/` - API REST con Node.js/TypeScript
- `frontend/` - Aplicación web cliente y panel admin (Next.js)
- `mobile/` - Aplicación móvil (React Native/Expo)
- `shared/` - Código compartido (tipos, validaciones)

## Características

- **2 Salas**: Roja y Amarilla
- **Panel Administrador**: Gestión de películas, funciones, precios, combos
- **Cliente Web/Móvil**: Cartelera, selección de asientos, compra de entradas y snacks
- **Kiosco**: Catálogo de productos configurables (pochoclos, bebidas, snacks)
- **Pagos**: Integración con Mercado Pago
- **Gestión de Asientos**: Bloqueo temporal, reservas, ocupación

## Tecnologías

- Backend: Node.js, TypeScript, Express, Prisma, PostgreSQL, Redis
- Frontend: Next.js, React, TypeScript, Tailwind CSS
- Mobile: React Native, Expo, TypeScript
- Pagos: Mercado Pago SDK
